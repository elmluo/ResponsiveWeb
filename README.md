#基于html5+css3的响应式页面布局
========================================

**1.项目前期预备知识：**
    * 项目没有用bootstrap，用原生基础方式实现，所以需要开发者用扎实的基础
    * html5和css3熟练掌握
    * javascript基础扎实，包括对jquery的运用。
    * 熟知响应式布局原理
    * 熟悉pc端和移动端调试
    * 熟悉gulp原理

**2.项目前端搭建：**
    * 使用html5新标签搭建页面框架
    * 使用css3实现样式预处理
    * 如果你对compass和sass掌握熟练，也可以用来管理样式代码，该部分在项目分支实现
    * 利用OwlCarousel2.js插件实现广告轮播效果
    * 实现广告的响应式图片，这里有些坑需要踩
    * 利用npm的brower-sync实现多浏览器端对页面兼容性调试。
    * gulp打包和发布代码
